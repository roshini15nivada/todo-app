# ✅ To-Do List App

A clean, functional To-Do list web app built with **HTML, CSS, and JavaScript**. Supports adding, completing, deleting, and filtering tasks.

## 🔗 Live Demo
[Try it here](https://roshini15nivada.github.io/todo-app)

---

## ✨ Features
- Add tasks by typing and pressing **Enter** or clicking **+**
- Mark tasks as **complete** with a checkmark button
- **Delete** individual tasks
- **Filter** tasks: All / Active / Done
- **Clear Completed** button to bulk remove done tasks
- Live count of remaining tasks
- Today's date shown in header
- Sample tasks preloaded so it doesn't look empty
- Smooth slide-in animation for new tasks

## 🛠️ Built With
| Technology | Purpose |
|---|---|
| HTML5 | App structure |
| CSS3 | Light purple theme, animations |
| JavaScript | Task logic, filtering, DOM updates |

1. Clone: `git clone https://github.com/roshini15nivada/todo-app.git`

- Dynamically rendering a list with `innerHTML`
- Filtering data and re-rendering the UI
